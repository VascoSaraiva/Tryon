
O projeto não está, de momento, alojado num servidor. Apenas poderá ser visualizado em localhost.

Para abrir o projeto é necessário ter o node instalado (https://nodejs.org/en/).

Depois de ter o node instalado:
    1- Entre na pasta Tryon Project através de um terminal.
    2- Instale as dependências do projeto através do comando - npm install
    3- Por fim, execute o comando - npm start - para abrir o projeto no browser em localhost.
